# twisher
A long twitter chain joiner

## Prerequsities
* Openjdk 8


## Running
```
  ./gradlew quarkusDev 
```

## parameters
|name | description |
|-----|-------------|
|twitter.access.token | twitter access token |
|twitter.access.token.secret  | twitter access token secret |
